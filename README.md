## Cara Menggunakan

Setiap folder berisi:

- `README.md` - Penjelasan lengkap dalam Bahasa Indonesia
- `main.go` - Kode demonstrasi
- File pendukung lainnya sesuai kebutuhan

## Prasyarat

- Go 1.22 atau lebih baru
- PostgreSQL (untuk demonstrasi database)
- Postman atau curl (untuk testing API)

## Menjalankan Demo

```bash
# Masuk ke folder demo
cd 01-hello-world

# Jalankan program
go run main.go
```
